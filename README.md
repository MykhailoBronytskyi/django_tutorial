### Commands

#### Install django
```
pip install django
python -m django --version
```
#### Create project and startserver
```
django-admin
django-admin startproject name
python manage.py runserver
```
* HTTP addresses:
    * http://127.0.0.1:8000/
    * http://localhost:8000/

#### Create app
```
python manage.py startapp name
```
* HTTP addresses:
    * http://127.0.0.1:8000/blog/
    * http://127.0.0.1:8000/blog/about/
